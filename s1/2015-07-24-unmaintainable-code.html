Naming

Single Letter Variable Names

  • Exemple :
    • a, b, c
    • i, j, k
    • ii, jj, kk
  • Avantages :
    • ne veut rien dire
    • recherche impossible
    • on utilise i, j, k depuis FØRTRAN. Les traditions c'est important

New Uses For Names For Baby

  • Exemple :
    • $jean->louis($david)
    • un prénom tappé sur un clavier bepo :)

Creative Miss-spelling

  • Exemple :
    • $somg
    • _contruct()
  • Avantages :
    • complique la recherche

Be Abstract

  • Exemple :
    • routineX48
    • HandleStuff
    • PerformDataFunction
    • do_args_method

A.C.R.O.N.Y.M.S.

Use acronyms to keep the code terse. Real men never define acronyms; they understand them genetically.

Misleading names

Make sure that every method does a little bit more (or less) than its name suggests. As a simple example, a method named isValid(x) should as a side effect convert x to binary and store the result in a database.

Hungarian Notation

Hungarian Notation is the tactical nuclear weapon of source code obfuscation techniques; use it!

Documentation

  • cacher du code dans les commentaires
  • documenter COMMENT mais pas POURQUOI
  • préciser quand c'est évident /* add 1 to i */
  • mentir dans les commentaires (meme par omission)
  • soyez exhaustif, une doc doit être la plus complete possible, l'algo important doit être détaillé dans la section 1.2.4.6.3.13
  • ne précisez jamais les unités de mesure (un int c'est un int)
  • ne jamais commenter une variable

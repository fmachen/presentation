JSON VS XML

Dois je faire du JSON ou du XML ?

Présentation de Florent Machen pour Recisio le 15/01/2016

Présentation

A ma gauche JSON

  • JavaScript Object Notation
  • open standard format : RFC 7159 and ECMA-404
  • deux types d'éléments structurels
    • paires nom / valeur
    • listes ordonnées de valeurs
  • objets, tableaux, booléen, nombre, chaîne, null

A ma droite XML

  • Extensible Markup Language
  • Ensemble de norme qui se combinent entre elles
  • Extensible donc une infinitée de type
    • validable par un schéma
    • XML transformable en un autre XML

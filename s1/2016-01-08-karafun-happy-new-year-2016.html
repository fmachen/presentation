Global State

  • It is hard to test code coupled strongly with a global variable and cannot control creation.
    (change to a global variable might break another test)
  • What's a global variable ?
    • global variable ($GLOBALS['variable'], global $variable)
    • super global ($_ENV, $_POST, $_GET, $_COOKIE, $_SERVER)
    • static
    • singleton

Risky Tests

  • Useless Tests (--report-useless-tests)
  • Unintentionally Covered Code (--strict-coverage)
  • Output During Test Execution (--disallow-test-output)
  • Test Execution Timeout (--enforce-time-limit)
  • Global State Manipulation (--strict-global-state)

Incomplete and Skipped Tests

  • $this->markTestIncomplete('This test has not been implemented yet.');
  • $this->markTestSkipped('The MySQLi extension is not available.');
  • Skip using @requires

Déjà constaté

  • Donne un cas d'utilisation avant de coder
  • Donne envie/l'occasion de revoir le code
